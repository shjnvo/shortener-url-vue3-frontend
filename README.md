# README

This Rails app is an implementation of a bit.ly like url shortener.

## Technologies

* Node 12.19.0
* Vue3
* Bosstrap 4

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Recommend use Docker for development
```
docker-compose up   
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
